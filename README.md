# Silk

A cross-platform chat application built with Kotlin Multiplatform.

## Quick Start

<!-- export JAVA_HOME=/usr/lib/jvm/java-17-openjdk -->
<!-- export PATH=$JAVA_HOME/bin:$PATH -->

# Start backend
# export JAVA_HOME=/usr/lib/jvm/java-17-openjdk && export PATH=$JAVA_HOME/bin:$PATH && cd /mi/sfs_turbo/lilin_v1/code/silk && ./gradlew :backend:build --no-daemon

./gradlew :frontend:webApp:browserDevelopmentRun
